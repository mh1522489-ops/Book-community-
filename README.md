## 🧠 فكرة المشروع

**Tech Book Club** هو موقع لعرض نادي قراءة افتراضي متخصص في الكتب التقنية.  
يهدف المشروع إلى تقديم تجربة جذابة بصريًا وسهلة الاستخدام، مع التركيز على:

- تشجيع المطورين على القراءة الجماعية
- عرض خطوات الانضمام إلى النادي
- تقديم خيارات عضوية مرنة (Starter / Pro)
- بناء مجتمع تقني تفاعلي

---

## ✨ الميزات الرئيسية

| الميزة | الوصف |
|--------|--------|
| 🎨 **تصميم عصري** | واجهة نظيفة بألوان متناسقة (أخضر غامق & ذهبي دافئ) |
| 📱 **متجاوب بالكامل** | يعمل بشكل مثالي على الهواتف، الأجهزة اللوحية، والحواسيب |
| 🃏 **بطاقات عضوية** | خيارات اشتراك واضحة مع أسعار ومميزات |
| ✅ **قائمة تفاعلية** | Checkboxes لعرض مزايا النادي |
| 🖼️ **معرض صور جذاب** | صور عالية الجودة من Pexels تعزز الهوية البصرية |
| 🧭 **هيكل منطقي** | أقسام مرتبة (عنوان، مميزات، رحلة العضو، خطط الأسعار) |
| ✨ **تأثيرات Hover** | تحسين تجربة المستخدم مع حركات بسيطة على الأزرار والبطاقات |

---

## 🛠️ التقنيات المستخدمة

- **HTML5** – هيكل الصفحة ومعايير الوصول
- **CSS3** – التنسيقات المتقدمة (Flexbox، Grid،渐变، الظلال)
- **Google Fonts (Inter)** – خط احترافي ومقروء
- **Pexels API (صور)** – محتوى بصري عالي الجودة
- **تصميم بدون مكتبات خارجية** – كود خفيف ونقي 100%
